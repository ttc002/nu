# Линейный и нелинейный matching пучка в инжекционном канале

## Название
Двухэтапный matching параметров пучка в канале с диполями, квадруполями и skew-коррекцией: сравнение 6x6-формализма и нелинейного трекинга

## Авторы
 - Шокал П.С
 - Вдовиченко П.А
 - Мельниченко Д.А
 - Токарев В.К.

## Аннотация
В работе исследуется транспорт пучка частиц в канале с поворотом на 90° с использованием двух подходов: линейного (6x6 матричного) и нелинейного (макрочастичный трекинг). Реализован двухэтапный matching: (1) грубый поиск рабочей точки в линейной модели, (2) локальный тонкий подбор в нелинейной модели. 


---

## 1. Введение
### 1.1 Мотивация
Требование к каналу: доставить пучок от начального состояния к целевым параметрам на входе в следующий узел ускорителя при сохранении малой дисперсии. На практике оптимизация должна учитывать как линейную оптику, так и нелинейные и хроматические искажения.

### 1.2 Цель работы
Подобрать и проверить сущестующие модели инжекторных труб для кольцевых ускорителей, смоделировать некоторые из них чтобы выбрать лучшую под нашу задачу

## 3. Модель канала и методы
### 3.1 Состав канала
- 3 секторных диполя по 30° (с краевой фокусировкой)
- 4 основных квадруполя
- matching-секция из trim-квадруполей
- 2 skew-квадруполя
- 2 секступоля
- выходной дрейф (позиция точки matching)

### 3.2 Линейная модель
Фазовый вектор: $\mathbf{X} = (x, p_x, y, p_y, z, \delta)^T$.

Транспорт:
$$
\mathbf{X}_{out} = R\,\mathbf{X}_{in}, \qquad
\Sigma_{out} = R\,\Sigma_{in}\,R^T.
$$

Twiss-параметры извлекаются из 2x2 блоков ковариационной матрицы.

### 3.3 Нелинейная модель
- Пошаговый symplectic-like drift-kick-drift для квадруполей/секступолей.
- Нелинейная кривизна в диполях, краевые кики.
- Оценка выходных параметров по выборочной ковариации макрочастиц.

### 3.4 Двухэтапная оптимизация
1. **Coarse stage (линейный)**: multistart matching по линейной целевой функции.
2. **Fine stage (нелинейный)**: локальная подстройка в узком диапазоне по нелинейной цели.
